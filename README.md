# forest-fire-detection-thermal-images
This project detects forest fires using thermal images and deep learning, identifying heat patterns for faster and more accurate detection even in low visibility conditions.

## 📌 Abstract
Forest fires pose a serious threat to ecosystems, wildlife, and human life. Early detection is crucial to minimize damage and ensure timely response. This project uses thermal imaging and deep learning techniques to detect forest fires accurately, even in low visibility conditions such as smoke, fog, or nighttime.

## 🎯 Objectives
- Develop an automated fire detection system  
- Use thermal images to detect heat patterns  
- Apply CNN for accurate classification  
- Provide real-time results through a web interface  

## 🚀 Features
- Fire detection using thermal images  
- Fast and accurate predictions  
- Works in low visibility conditions  
- Simple web-based interface  
- Real-time output  

## 🛠️ Tech Stack

## 🧠 Methodology
1. Collect thermal image dataset  
2. Preprocess images  
3. Train CNN model  
4. Predict fire or non-fire  
5. Display results via web app
